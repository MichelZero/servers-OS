Instale o EVE-NG Community no VirtualBox

O seu guia passo a passo para criar um laboratório de redes poderoso e gratuito.

O que é o EVE-NG?

O EVE-NG (Emulated Virtual Environment - Next Generation) é uma plataforma de emulação de redes que permite a criação de laboratórios complexos com dispositivos de múltiplos fornecedores (Cisco, Juniper, Fortinet, Check Point, etc.). É uma ferramenta essencial para estudantes, engenheiros de rede e profissionais de TI que precisam testar configurações, estudar para certificações ou simular ambientes de produção.

Pré-requisitos

Antes de começar, garanta que você tem o seguinte:

  • Oracle VirtualBox: Instalado e funcionando no seu computador.
  • Hardware com Virtualização: A tecnologia de virtualização (Intel VT-x ou AMD-V) deve estar ativada na BIOS/UEFI do seu computador.
  • Recursos Mínimos: Pelo menos 8 GB de RAM e 40 GB de espaço em disco livre são recomendados para um bom desempenho.
  • Conexão com a Internet: Para baixar os arquivos necessários.

Escolha o seu Método de Instalação

Existem duas formas principais de instalar o EVE-NG no VirtualBox. Recomendamos o Método 1 (via OVF) por ser mais rápido e simples.

Método 1: Instalação via OVF (Recomendado)

Este método utiliza uma máquina virtual pré-configurada (formato OVF) fornecida diretamente pelo time do EVE-NG.

  1. Baixe o arquivo OVF: Acesse a página de downloads do EVE-NG e baixe a versão "EVE-NG Community Edition OVF" para "Bare EVE on VMWare". Não se preocupe, o arquivo funciona perfeitamente no VirtualBox.
  2. Importe a Appliance no VirtualBox:
    • Abra o VirtualBox.
    • Vá em Arquivo > Importar Appliance....
    • Na janela que abrir, selecione o arquivo .ovf que você baixou e clique em "Próximo".
    • Revise as configurações da VM (pode aumentar a RAM e o número de CPUs se desejar) e clique em "Importar".
  3. Ajuste a Rede:
    • Com a nova VM (EVE-NG) selecionada, vá em Configurações > Rede.
    • No Adaptador 1, mude a opção "Conectado a" para "Placa de Rede em Modo Bridge". Isso fará com que sua VM pegue um IP da sua rede local, facilitando o acesso.
    • Clique em "OK".
  4. Inicie a VM: Inicie a máquina virtual EVE-NG. Uma tela de console preta aparecerá.
  5. Acesse o EVE-NG: Após a inicialização, a tela de login mostrará o endereço IP da máquina virtual. Anote este IP.
    • Abra seu navegador e acesse http://[ENDEREÇO_IP_DA_VM].
    • As credenciais padrão são: Usuário: admin / Senha: eve.
    • Pronto! Você está no painel do EVE-NG.
Nota: As credenciais de login para o console (tela preta) são: Usuário: root / Senha: eve.

Método 2: Instalação em um Servidor Ubuntu (Avançado)

Este método oferece mais flexibilidade, pois você instala o EVE-NG sobre uma instalação limpa do Ubuntu Server.

  1. Baixe o Ubuntu Server: Baixe a imagem ISO do Ubuntu Server 22.04 LTS.
  2. Crie uma Nova VM no VirtualBox:
    • Clique em "Novo".
    • Nomeie a VM (ex: EVE-NG Ubuntu), selecione a imagem ISO do Ubuntu que você baixou.
    • Marque a opção "Skip Unattended Installation".
    • Aloque recursos: no mínimo 4GB de RAM, 2 CPUs e 40GB de disco.
    • Finalize a criação da VM.
  3. Configure a Rede: Antes de iniciar, vá em Configurações > Rede e mude o Adaptador 1 para "Placa de Rede em Modo Bridge".
  4. Instale o Ubuntu Server: Inicie a VM e siga o assistente de instalação do Ubuntu.
    • Selecione o idioma e layout de teclado.
    • Na configuração de rede, deixe o padrão (DHCP).
    • Use o disco inteiro para a partição.
    • Defina seu nome, nome do servidor, nome de usuário e senha.
    • Importante: Na etapa "SSH Setup", marque a opção "Install OpenSSH server".
    • Não instale nenhum software adicional (snaps). Aguarde a instalação terminar e reinicie a VM.
  5. Instale o EVE-NG: Após o login no console do Ubuntu, execute o seguinte comando para baixar e executar o script de instalação automática do EVE-NG. 
    wget -O - https://www.eve-ng.net/focal/install-eve.sh | bash -i

    O script fará todo o trabalho. Ao final, o sistema será reiniciado.

  6. Acesse o EVE-NG: Após reiniciar, a tela de login mostrará o endereço IP. Use esse IP no seu navegador para acessar a interface web com as credenciais padrão (admin/eve).

Próximos Passos

Parabéns! Seu laboratório EVE-NG está pronto. O próximo passo é o mais importante: adicionar as imagens dos sistemas operacionais (como Cisco IOS, ASA, Juniper Junos, etc.) que você deseja usar em seus laboratórios.

Este processo envolve o upload de arquivos de imagem (QEMU, IOL, Dynamips) para diretórios específicos dentro da VM do EVE-NG. Consulte a documentação oficial do EVE-NG para obter instruções detalhadas sobre como adicionar cada tipo de imagem.
