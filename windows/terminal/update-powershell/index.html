Atualize o PowerShell sem Sair do VS Code

Recebeu a notificação de PowerShell desatualizado? Resolva isso em 60 segundos com um único comando no seu terminal.

Método Recomendado: Atualização via Winget

O Winget é o gerenciador de pacotes oficial do Windows, já integrado ao sistema. É a forma mais limpa e moderna de gerenciar seus aplicativos via linha de comando.

Passo 1: Abra o Terminal no VS Code

Use o atalho Ctrl + ' para abrir o painel do terminal. Certifique-se de que a aba selecionada é o PowerShell.

Passo 2: Execute o Comando de Atualização

Copie e cole o comando abaixo no seu terminal e pressione Enter. Ele encontrará e atualizará o PowerShell para a versão mais recente.

winget upgrade Microsoft.PowerShell

Passo 3: Reinicie o Terminal e Verifique

Após a instalação ser concluída, feche e reabra o terminal (clicando no ícone da lixeira). A sessão antiga não usará a nova versão. Para confirmar, execute:

$PSVersionTable.PSVersion

A saída deverá mostrar a nova versão que você acabou de instalar.

Alternativa: Usando Chocolatey

Se você já é um usuário do gerenciador de pacotes Chocolatey, o comando é igualmente simples.

choco upgrade powershell -y

Ponto Importante: Permissões de Administrador

Para instalar ou atualizar software, o terminal precisa de privilégios elevados. Se o comando falhar com um erro de acesso negado, você precisará:

  1. Fechar completamente o VS Code.
  2. Clicar com o botão direito no ícone do VS Code.
  3. Selecionar "Executar como administrador".
  4. Repetir o comando de atualização no terminal.

Mantenha seu ambiente de desenvolvimento moderno e eficiente. Um terminal atualizado é o primeiro passo.
