Welcome to RouterOS! (Recomendações)

Uma lista de boas práticas de segurança que você deve seguir:

  1. Definir uma senha forte para o roteador.
  2. Atualizar o software (RouterOS) para a versão mais recente.
  3. Manter o firewall ativado para redes não confiáveis.
  4. Definir uma senha forte e personalizada para o Wi-Fi.
  5. Configurar seu país para seguir as regulamentações de rádio locais.

RouterMode (Modo do Roteador)

Descreve a lógica principal da configuração:

  • A porta WAN (internet) está protegida por firewall e configurada para receber IP automaticamente.
  • As interfaces Wireless e Ethernet (exceto a WAN) estão agrupadas em uma ponte (bridge), funcionando como uma única rede local (LAN).

LAN Configuration (Rede Local)

Detalhes da sua rede interna:

  • Endereço IP do roteador: 192.168.88.1/24
  • Servidor DHCP: Ativado (distribui IPs automaticamente).
  • Servidor DNS: Ativado (permite a navegação por nomes).

Ações Disponíveis (Botões)

Remove Configuration

Apaga completamente a configuração padrão, deixando o roteador "em branco". Use com cuidado, pois o acesso via IP será perdido.

Show Script...

Exibe os comandos exatos do RouterOS usados para criar esta configuração. Ótimo para aprendizado.

OK

Ação mais comum. Fecha esta janela informativa e permite que você comece a usar o roteador com a configuração padrão ativa.
