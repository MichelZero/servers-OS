Firmware 6B2QJXD7 para o Samsung 990 Pro

Lançado em 02 de julho de 2025

Atualização de Estabilidade Crítica para o seu SSD

A Samsung lançou uma nova atualização de firmware, versão 6B2QJXD7, para a sua aclamada linha de SSDs 990 Pro. Esta atualização é crucial para corrigir problemas específicos que alguns usuários vinham enfrentando, garantindo maior estabilidade e confiabilidade para o seu sistema.

O que há de novo na versão 6B2QJXD7?

Correção Principal

De acordo com as notas de lançamento, esta atualização foi desenvolvida especificamente para resolver um problema crítico: a não detecção intermitente do SSD e a ocorrência de tela azul (BSOD). [2] [3] Se o seu sistema reiniciava e não reconhecia o drive, ou apresentava falhas inesperadas, esta atualização é essencial, conforme discutido por vários usuários.

Como Atualizar seu 990 Pro

A maneira mais fácil e recomendada de atualizar o firmware é através do software Samsung Magician.

  1. Backup dos Dados: Antes de qualquer atualização, é altamente recomendável fazer um backup completo dos seus dados, uma prática padrão para segurança.
  2. Abra o Samsung Magician: Certifique-se de ter a versão mais recente do software, disponível no site oficial da Samsung.
  3. Vá para a Aba de Atualização: No menu lateral, clique em "Update". O Magician irá verificar automaticamente por um novo firmware.
  4. Inicie a Atualização: Selecione o seu SSD 990 Pro na lista e clique no botão "Update".
  5. Reinicie o Sistema: Após a conclusão, o software pedirá para você reiniciar o computador para que as alterações entrem em vigor.

Como alternativa, a Samsung também disponibiliza um arquivo ISO de boot para atualização via DOS, que pode ser encontrado na mesma página oficial de downloads.

Possíveis Problemas na Instalação e Soluções

Alguns usuários relataram dificuldades durante o processo de atualização. Aqui estão as soluções mais comuns:

Falha na Atualização pelo Magician

Problema: O Samsung Magician exibe a mensagem "Failed to update firmware on the selected drive".

Solução:

  • Executar como Administrador: Feche o Magician, clique com o botão direito no ícone e selecione "Executar como administrador". Esta é a solução mais eficaz relatada por usuários.
  • Atualizar um Drive por Vez: Se você possui múltiplos SSDs 990 Pro, tente atualizá-los individualmente.
  • Modo de Energia: Altere o plano de energia do Windows para "Alto Desempenho" antes de tentar novamente, pois modos de economia podem interferir na atualização.
