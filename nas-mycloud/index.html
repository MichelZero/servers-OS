Guia de Acesso SSH

Passo 1: Habilitando o Acesso SSH

Primeiro, ative o serviço SSH no painel de controle do seu My Cloud. Vá para Configurações > Rede. Na seção "Serviços de Rede", localize e ative a opção SSH. Será solicitado que você crie e confirme uma senha segura para o acesso.

Passo 2: Conectando-se via SSH

Com o SSH ativado, use um cliente SSH (como o Terminal no macOS/Linux ou o PuTTY/PowerShell no Windows) para se conectar.

Para macOS e Linux:

Abra o Terminal e digite o comando:

ssh sshd@192.168.0.10

Para Windows (PowerShell/CMD):

Abra o PowerShell e use o mesmo comando:

ssh sshd@192.168.0.10

Ao se conectar pela primeira vez, você precisará aceitar a chave do host. Em seguida, insira a senha que você configurou no Passo 1.

Solução de Problemas Comuns

Se a conexão for recusada por causa de algoritmos de chave de host, tente usar o seguinte comando para permitir o algoritmo `ssh-dss`:

ssh -oHostKeyAlgorithms=+ssh-dss sshd@192.168.0.10
