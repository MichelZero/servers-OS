Configurando Comunicação Direta de 2.5Gb no Proxmox

Para configurar a comunicação entre seus dois nós Proxmox (pve-01 e pve-02) utilizando as placas de rede enp3s0 de 2.5Gb, você criará uma nova ponte (Linux Bridge) em cada nó. Isso estabelece um link dedicado de alta velocidade, ideal para comunicação de cluster, replicação e migrações.

Configuração do Nó pve-01

  1. Acesse a interface web do Proxmox.
  2. Navegue para: Datacenter → pve-01 → Sistema → Rede.
  3. Clique em Criar e selecione "Ponte Linux" (Linux Bridge).
  4. Preencha os campos da seguinte forma:
    • Nome: vmbr1
    • Endereço IPv4/CIDR: 10.10.10.1/24
    • Portas da ponte: enp3s0
    • Gateway: Deixe este campo em branco.
    • Marque a opção "Autostart".
  5. Clique em Criar para finalizar.

Configuração do Nó pve-02

  1. Acesse a interface web do Proxmox.
  2. Navegue para: Datacenter → pve-02 → Sistema → Rede.
  3. Clique em Criar e selecione "Ponte Linux" (Linux Bridge).
  4. Preencha os campos da seguinte forma:
    • Nome: vmbr1
    • Endereço IPv4/CIDR: 10.10.10.2/24
    • Portas da ponte: enp3s0
    • Gateway: Deixe este campo em branco.
    • Marque a opção "Autostart".
  5. Clique em Criar para finalizar.

Aplicar as Configurações e Verificar

  1. Após criar as pontes em ambos os nós, um botão "Apply Configuration" aparecerá no topo. Clique nele em cada nó para aplicar as novas configurações.
  2. Para verificar a comunicação, acesse o shell de cada nó e use o comando ping:

No shell do pve-01, execute:

ping 10.10.10.2

No shell do pve-02, execute:

ping 10.10.10.1

Se os pings forem bem-sucedidos, a comunicação direta pela rede de 2.5Gb estará funcionando corretamente.
