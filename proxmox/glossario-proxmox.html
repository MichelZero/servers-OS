Glossário de Termos Importantes - Proxmox

  • Proxmox VE: Plataforma de virtualização open source baseada em Debian.
  • VM (Máquina Virtual): Ambiente virtualizado que simula um computador físico.
  • Container: Ambiente isolado para aplicações, mais leve que VMs.
  • Passthrough: Técnica para repassar hardware físico (GPU, USB, etc) diretamente para uma VM.
  • IOMMU: Tecnologia que permite o passthrough de dispositivos PCI para VMs.
  • VT-x/VT-d: Tecnologias Intel para virtualização e passthrough.
  • SVM: Tecnologia AMD para virtualização.
  • Barcelo: Família de processadores AMD com iGPU, requer atenção especial para passthrough.
  • Secure Boot: Mecanismo de segurança da BIOS/UEFI que pode impedir instalação de sistemas alternativos.
  • ISO: Imagem de disco utilizada para instalação do sistema.
  • Storage: Sistema de armazenamento utilizado pelo Proxmox para VMs e containers.
  • Cluster: Conjunto de servidores Proxmox trabalhando em conjunto.
  • Bridge: Interface de rede virtual utilizada para conectar VMs à rede física.
