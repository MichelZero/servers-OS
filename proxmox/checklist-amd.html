Checklist de Instalação - AMD

  • Verificar suporte a virtualização (SVM/IOMMU)
  • Atualizar BIOS para versão mais recente
  • Desabilitar Secure Boot
  • Configurar boot pelo pendrive/ISO
  • Instalar Proxmox seguindo o assistente
  • Configurar rede e armazenamento
  • Para processadores com iGPU Barcelo: configurar passthrough conforme documentação oficial
  • Testar criação de VMs e passthrough de GPU
