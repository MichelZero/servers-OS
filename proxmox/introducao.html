Introdução

O Proxmox Virtual Environment (Proxmox VE) é uma plataforma de virtualização open source baseada em Debian, amplamente utilizada para criar e gerenciar máquinas virtuais (VMs) e containers de forma eficiente e segura. Ele oferece uma interface web intuitiva, recursos avançados de gerenciamento, suporte a alta disponibilidade, backup, snapshots e integração com diversos tipos de armazenamento.

Este guia foi elaborado para ajudar iniciantes e profissionais a realizar a instalação do Proxmox VE, abordando desde os requisitos básicos até dicas específicas para processadores Intel e AMD, incluindo cuidados especiais para quem utiliza processadores AMD Barcelo com iGPU e deseja realizar passthrough de GPU.

Ao seguir este material, você aprenderá:

  • O que é o Proxmox VE e suas principais vantagens
  • Como preparar o hardware e o ambiente para instalação
  • Passo a passo para instalar o Proxmox em sistemas Intel e AMD
  • Cuidados e problemas comuns durante a instalação
  • Glossário dos principais termos utilizados no universo Proxmox

O Proxmox é ideal para laboratórios, ambientes de produção, homelabs e pequenas empresas que buscam uma solução robusta, flexível e sem custos de licença.
